== Description ==

Customize Twenty Sixteen theme - add Google Fonts, use new templates without sidebar and get other options to easily customize your site. 

This plugin simplifies customization of Twenty Sixteen theme by giving you additional Customizer options including the usage of Google Fonts, and usage of additional page templates.

= New Templates =

Customize Twenty Sixteen adds additional page templates which can be selected on page edit:

1. No sidebar boxed (default boxed, but page without sidebars)
2. No sidebar wide (wide page without sidebars)
3. No sidebar fullscreen (use with Bold Builder)

= Customization Options =

Also it will add new customization section (BoldThemes Settings) with following options:

1. Body font
2. Title font
3. Heading font
4. Menu font
5. Remove space around pages
6. Remove space around header
7. Remove entry header
8. Menu position - default, left and centered below logo
9. Custom CSS
10. Custom JS

Options allow you to change the visual appearance of pages and add custom JavaScript and CSS code without creating a child theme or editing theme files. Easily add Google Analytics, Facebook Pixel or similar without theme files modification.

== Installation ==

Use WordPress Add New Plugin feature and search Customize Twenty Sixteen or download the archive and:

1. Unzip the archive on your computer
2. Upload bt_customize_2016 directory to the /wp-content/plugins/ directory
3. Activate the plugin through the Plugins menu in WordPress
